# Willkommen bei Postwerk

_Postwerk macht Ihren Posteingang programmierbar. Verbinden Sie ein Postfach, bauen Sie eine visuelle Automatisierung, und lassen Sie jede eingehende E-Mail automatisch lesen, weiterleiten und bearbeiten._

Eine **Automatisierung** ist ein Graph aus _Nodes_. E-Mails treten über einen Trigger ein, durchlaufen Verarbeitungs- und Entscheidungs-Nodes und enden in Aktionen wie Antworten, Beschriften oder dem Aufruf einer API. Sie bauen das auf einer Leinwand zusammen — ganz ohne Code — und Postwerk führt es bei jeder passenden Nachricht aus.

> [!TIP]
> Wenig Zeit? Springen Sie direkt zum [Schnellstart](/docs/getting-started/quickstart) und bringen Sie in etwa zehn Minuten eine lauffähige Automatisierung an den Start.

## Was Sie bauen können

- **Triage** — E-Mails nach Absicht klassifizieren und jede Art an die richtige Stelle leiten.
- **Extraktion** — Bestellnummern, Namen und Beträge aus Freitext herauslösen.
- **Antworten** — personalisierte, vorlagenbasierte Antworten sofort versenden.
- **Integrationen** — Daten an Ihr CRM, Ticketing-System oder jede HTTP-API übergeben.

## Drei Wege zum Start

| Weg | Am besten für | Wo |
|-----|---------------|----|
| Auf der Leinwand bauen | Volle Kontrolle über jeden Node | Automatisierungen → Neu |
| KI-Assistent fragen | Die Aufgabe in normaler Sprache beschreiben | Assistent → Workflow beschreiben |
| Aus dem Marketplace installieren | Bewährte Rezepte, die Sie anpassen können | Marketplace → Installieren |

Der visuelle Editor hat drei Bereiche: links die Werkzeugbibliothek, in der Mitte die Leinwand und rechts den Inspektor, in dem Sie den ausgewählten Node konfigurieren.
