# Welcome to Postwerk

_Postwerk turns your inbox into a programmable surface. Connect a mailbox, build a visual automation, and let every incoming email be read, routed, and acted on — automatically._

An **automation** is a graph of _nodes_. Mail enters through a trigger, flows through processing and decision nodes, and ends in actions like replying, labeling, or calling an API. You build it on a canvas — no code required — and Postwerk runs it on every matching message.

> [!TIP]
> In a hurry? Jump to the [Quickstart](/docs/getting-started/quickstart) to ship a working automation in about ten minutes.

## What you can build

- **Triage** — classify mail by intent and route each kind to the right place.
- **Extraction** — pull order numbers, names, and totals out of free text.
- **Replies** — send personalized, templated responses instantly.
- **Integrations** — push data to your CRM, ticketing, or any HTTP API.

## Three ways to start

| Path | Best for | Where |
|------|----------|-------|
| Build on the canvas | Full control over every node | Automations → New |
| Ask the AI assistant | Describing the job in plain language | Assistant → Describe a workflow |
| Install from Marketplace | Proven recipes you can tweak | Marketplace → Install |

The visual editor has three regions: the tool library on the left, the canvas in the middle, and the inspector on the right where you configure the selected node.
